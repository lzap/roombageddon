# ROOMBAGEDDON

A TIC-80 game by Lukáš Zapletal aka `lzap`.

## Requirements

- Linux or macOS (it probably works on Windows with MinGW or WSL)
- GNU Make
- TIC-80 PRO (consider buying the PRO version or sponsoring @nesbox)
- luarocks
- luacc
- selene
- stylua

### Runing

To run and test your game:

```
make run
```

### Building

To build your game:

```
make build
```
